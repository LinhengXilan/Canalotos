#### 描述

	在Source/KernelData.h中定义了内核将要使用的内存信息结构体:

	{
		UINTN Size;
		EFI_MEMORY_DESCRIPTOR* Buffer;
		UINTN MapKey;
		UINTN DescriptorSize;
		UINT32 DescriptorVersion;
	}EFI_DATA_MEMORY;
